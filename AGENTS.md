# TermArt Project Context

This project is a Rust-based terminal art and system monitoring application. It provides various visualizations including system metrics, ASCII art, and interactive terminal displays.

## Core Instructions

Load the following file for core project instructions:
@asha/CORE.md

## Project Structure

- `src/` - Main source code directory
  - `monitor/` - System monitoring modules (CPU, memory, disk, network, Docker, GPU)
  - `viz/` - Visualization modules (various terminal art implementations)
  - `main.rs` - Entry point
  - `config.rs` - Configuration management
  - `terminal.rs` - Terminal handling utilities

## Key Features

- System resource monitoring
- Terminal-based visualizations
- ASCII art and animations
- Pomodoro timer
- Weather display
- Folding@home client integration

## Development Guidelines

When working on this project:
1. Follow Rust best practices and idioms
2. Maintain the modular structure
3. Ensure terminal compatibility across platforms
4. Keep performance in mind for real-time monitoring
